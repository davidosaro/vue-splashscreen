# vue-elegant-button

## Project setup

```
npm install --save vue-splashscreen
```

or

```
yarn add vue-splashscreen
```

## Basic Usage

```
<template>
  <VueSplashScreen
    background="#fff"
    color="#000"
    text="Hello World"
  />
</template>
<script>
import VueSplashScreen from "vue-splashscreen";

export default {
  components:{
    VueSplashScreen
  }
}
</script>
```
